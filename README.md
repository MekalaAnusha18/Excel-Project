📊 Sales Performance Dashboard

📌 Project Overview

This dashboard provides a comprehensive analysis of sales performance across regions, product categories, customer segments, and shipping modes. It is built using Excel and visualizes key metrics to support business decisions.

💼 Business Impact

Identifies high-performing regions and products

Highlights loss-making categories and discount impact

Supports strategic decisions for marketing, logistics, and customer targeting

📈 Key Metrics

Total Sales: ₹22,97,201

Total Profit: ₹2,86,397

Average Profit Margin: 12%

Quantity Sold: 37,873 units

🚀 Features

Regional sales and profitability breakdown

Product-wise profit/loss analysis

Top customers and top-selling products

Discount vs. profit margin scatter plot

Shipping mode efficiency and profit contribution

📊 Usage Instructions

Open the Excel file and navigate to the dashboard sheet

Use slicers and filters to explore data by region, category, and customer

Review charts for actionable insights

📸 Visual Highlights

Pie chart of profit by shipping mode

Bar charts for top customers and products

Regional profitability matrix

Category-wise sales and profit comparison

🛠️ Technologies Used

Microsoft Excel (Pivot Tables, Charts, Slicers)

Data cleaning and transformation in Excel
